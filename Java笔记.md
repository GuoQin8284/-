## 常量

常量就是在程序运行过程中，其值不可以发生改变的量

常量分类：

​	A. 字符常量  “helloWorld”

​	B. 整数常量   12，-23，0

​	C. 小数常量 	12.45

​	D. 字符常量 	'a'，'c'，'0'

​	E. 布尔常量	true，false

​	F. 空常量	null
